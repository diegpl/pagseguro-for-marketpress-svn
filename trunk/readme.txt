Payment gateway plugin for Pagseguro Brazilian Credit Cards, for Marketpress.
Plugin de gateway de pagamento pelo Pagseguro, para MarketPress.Version: 1.0

== Description ==

Easiest then ever, is just upload plugin to plugins-gateway folder of marketpress and it will be ready.

Our sites:
http://wpsoftwares.com
http://wpsoft.com.br

== Installation ==

1. Upload `pagseguro.php and other on the root folder` to the `/wp-content/plugins/marketpress/marketpress-includes/plugins-gateway/` directory
2. And it is ready. Now you should to insert your tolken and email on payment settings panel of Marketpress plugin.
